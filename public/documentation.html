🚀 Getting Started

Welcome to CineWave! Follow these simple steps to start streaming your favorite movies and series.

Create an Account

Visit the website and click Register. Enter your name, email, and password. You can also sign up with Google for quick access.

Verify Your Email

Check your inbox for a verification email from CineWave. Click the link to verify your account.

Choose a Subscription Plan

Browse available plans (Basic, Standard, Premium) and select one that fits your needs. New users get a free trial!

Start Watching

Browse the catalog, pick a movie or series, and enjoy seamless streaming!

🌐 Hosting & Deployment

CineWave can be deployed to Vercel for free with full server-side rendering support.

Prerequisites

  • GitHub Account — Push your code to a GitHub repository
  • Vercel Account — Sign up free at vercel.com
  • MongoDB Atlas — Free M0 cluster at mongodb.com/atlas
  • Cloudinary — Free tier at cloudinary.com
  • Google Cloud Console — For OAuth credentials

Deploy to Vercel

Push Code to GitHub

Push your CineWave project to a GitHub repository. Make sure .env.local is in .gitignore.

Import Project on Vercel

Go to vercel.com/new → Click "Import Git Repository" → Select your CineWave repo.

Configure Environment Variables

In Vercel project settings → Environment Variables, add all the variables listed in the table below.

Deploy

Click Deploy. Vercel will build and deploy automatically. You'll get a URL like your-app.vercel.app.

Post-Deploy Setup

Update NEXTAUTH_URL and NEXT_PUBLIC_APP_URL to your Vercel domain. Add the Vercel URL to Google OAuth redirect URIs.

Post-Deployment Checklist

  1. MongoDB Atlas → Network Access → Add 0.0.0.0/0 to allow Vercel's dynamic IPs
  2. Google Cloud Console → Credentials → Add https://your-app.vercel.app/api/auth/callback/google to Authorized Redirect URIs
  3. Cloudinary → Settings → Upload → Ensure your unsigned preset allows uploads from your Vercel domain
  4. SSLCommerz → Update success/fail/cancel URLs to your Vercel domain
  5. Update env vars → Set NEXTAUTH_URL and NEXT_PUBLIC_APP_URL to your Vercel domain
  6. Redeploy → After changing env vars, trigger a redeployment from Vercel dashboard
⚠️ Never commit .env.local to Git! It contains sensitive credentials. Use .env.local.example as a template.

Custom Domain (Optional)

To add a custom domain:

  1. Vercel → Project Settings → Domains
  2. Add your domain (e.g. cinewave.com)
  3. Configure DNS records as instructed by Vercel
  4. Update NEXTAUTH_URL and NEXT_PUBLIC_APP_URL to the custom domain
  5. Update Google OAuth redirect URIs to use the custom domain

❓ Frequently Asked Questions

How do I cancel my subscription?

Go to Account → Subscription and click "Cancel". Your access continues until the end of the billing period.

Can I watch on multiple devices?

Yes! The number of simultaneous devices depends on your plan — Basic (1), Standard (2), Premium (4).

Why can't I watch a movie/series?

You need an active subscription. If you see a "View Plans" message, your subscription may have expired.

How do I change my password?

Go to the login page → Click "Forgot Password" → Enter your email → Follow the reset link in your inbox.

Is my watch progress saved?

Yes! CineWave automatically saves your progress every 30 seconds. Resume watching from where you left off.

Can I download content for offline viewing?

Offline downloads are not available currently. Content is streamed directly in your browser.
